# Levelbook

Levelbook is a family of open-source, self-hostable field-records tools for construction. Tag it, scan it, capture it, keep the record.

Each tool handles one part of the job and leaves behind a durable, portable record you own. All four are AGPLv3, offline-capable, and built the same way: a PocketBase binary plus a static frontend. Self-hosting is a first-class path, not an afterthought.

## Products

| Product | What it does | Status |
| --- | --- | --- |
| [TrenchNote](https://github.com/levelbookdev/trenchnote) | Equipment and materials ledger | **Active** — v1.0 |
| [LoopCheck](https://github.com/levelbookdev/loopcheck) | Commissioning checkout | **Active** — pre-release |
| [SlipSign](https://github.com/levelbookdev/slipsign) | Same-day signed T&M tickets | **Stable** — v1.0, maintenance |
| [MainLine](https://github.com/levelbookdev/mainline) | Station-based pipeline records | **Dormant** — feature-complete for shipped phases |
