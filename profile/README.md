# Levelbook

Levelbook is a family of open-source, self-hostable field-records tools for construction. Tag it, scan it, capture it, keep the record.

Each tool handles one part of the job and leaves behind a durable, portable record you own. Self-hosting is a first-class path, not an afterthought: run the full stack on your own infrastructure.

- Website: https://levelbook.dev
- Public repositories: https://github.com/levelbookdev
